# ☕ Java #001 - Introduction to Java

## Q1. What is Java?

Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems and currently maintained by Oracle.

---

## Q2. Who developed Java?

Sun Microsystems.

---

## Q3. Who owns Java now?

Oracle Corporation.

---

## Q4. What is WORA?

Write Once, Run Anywhere.

---

## Q5. Why is Java platform independent?

Because Java programs are compiled into bytecode, which runs on the Java Virtual Machine (JVM) available on different operating systems.
